# Contributing

Contributions are welcome when they improve the Skill's research quality, traceability, or terminology discipline.

Useful contributions include:

- Clarifying concept and translation boundaries.
- Adding evidence templates or public-source methods.
- Adding investment-research example skeletons.
- Adding eval cases and counterexamples.
- Correcting mistranslations.
- Strengthening offline scripts.
- Improving the Prompt Pack or scorecard.

## Contribution Rules

1. Keep conclusions traceable to public sources and include dates.
2. Separate facts, source claims, project inferences, and open questions.
3. Include counterevidence, risks, tracking indicators, and invalidation conditions.
4. Run `python3 scripts/validate_skill.py` before submitting changes.
5. Keep scope narrow and explain terminology changes.

The project does not accept stock trade recommendations, return promises, non-public information, unauthorized content, personality simulation, or conclusions without traceable evidence.
