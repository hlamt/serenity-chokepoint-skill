# Distillation Rules

Use these rules for any future distillation of publicly available Serenity expressions.

1. Use public materials only.
2. Preserve source title, publisher or account, URL, publication date, and access date when relevant.
3. Separate original expression, faithful paraphrase, project inference, and terminology calibration.
4. Do not perform personality simulation or claim to speak for Serenity.
5. Make every methodology judgment traceable to evidence.
6. Do not infer a stable framework from one isolated statement.
7. Record counterexamples, changes over time, and uncertainty.
8. Respect copyright: quote only what is necessary and prefer concise paraphrase.
9. Do not use private posts, cookies, sessions, deleted material obtained without authorization, or paid material outside its license.

## Suggested Record

| field | description |
| --- | --- |
| source_id | Stable local identifier |
| source_title | Public title or short description |
| source_url | Public locator |
| published_at | Source publication date |
| source_expression | Minimal quote or faithful paraphrase |
| project_inference | What the project infers and why |
| terminology_calibration | Any Chokepoint / TOC boundary note |
| confidence | Current confidence and evidence gaps |
