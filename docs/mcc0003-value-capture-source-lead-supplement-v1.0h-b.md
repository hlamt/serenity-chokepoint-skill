1. operating bottleneck / chokepoint;
2. firm-level economics and value capture;
3. common-shareholder capture through equity appreciation; and
4. dilution, float dynamics, financing quality, valuation, priced-in timing, and structurally capped upside.

The operating constraint, the firm's ability to capture economics, and the common shareholder's ability to capture value are separate review layers. Evidence at one layer does not establish the others.
