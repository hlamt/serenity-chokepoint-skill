# Invalidation / Bear Case / Risk Source Lead Supplement — V1.0H-C

## 1. Purpose and boundary

This supplement organizes user-confirmed X URLs and bounded screenshot-visible signals concerning uncertainty, devil's-advocate review, bear cases, risk, false-positive control, and thesis-failure conditions.

It supplements MCC-0005 source-lead coverage and prepares a bounded input set for V1.0I. It does not independently open or verify posts, approve source records, validate method claims, produce TOC calibration, determine real-company outcomes, or provide investment advice.

- This supplement does not validate MCC-0005.
- This supplement does not create source records.
- This supplement does not approve runtime use.
- This supplement does not generate investment advice.

## 2. Input basis

The sole new input is user-provided stable X URLs, stated visible dates, and short screenshot-visible signals. These signals are review aids, not full post reproductions. External quotes and company, financing, market, regulatory, capacity, customer, valuation, and supply-chain assertions remain unverified.

No X search, scraping, API call, login, or thread navigation occurred in this round. Long posts, replies, and surrounding contexts require later URL-level review. The AXTI URL also appears in V1.0H and is cross-referenced here for its uncertainty and devil's-advocate role; it is not duplicate-approved.

## 3. MCC-0005 review frame

MCC-0005 concerns source governance, evidence sufficiency, falsification discipline, and overclaim control.

For this supplement, the relevant review dimensions are:

1. thesis uncertainty;
2. could-be-wrong framing;
3. devil's-advocate review;
4. bear-case review;
5. risk framing;
6. false-positive control;
7. dependency / beneficiary / local bottleneck versus active constraint distinction; and
8. capacity expansion, substitution, vertical integration, regulation, or market structure as thesis-failure conditions.

Uncertainty language alone does not constitute falsification discipline. A useful invalidation candidate should identify what evidence would weaken a thesis, which layer is being challenged, and whether the proposed risk changes the operating constraint, firm economics, or shareholder capture.

The bounded conclusion of this supplement is:

**MCC-0005 candidate support is materially strengthened at the source-lead level.**

MCC-0005 is not validated.
