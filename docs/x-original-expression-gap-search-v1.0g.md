# Targeted X Original Expression Gap Search v1.0G

## 1. Purpose

This file generates targeted X search locators and candidate source leads for the method-claim review gaps identified in `docs/method-claim-review-v1.0f.md`.

Scope boundaries:

- This is not corpus.
- This is not source records.
- This is not method claims.
- This is not validated claims.
- This is not TOC calibration.
- This is not investment advice.

This pass is limited to targeted locator construction and review-ready source-lead scouting around the V1.0F source gaps. X search locators are not source records. X search results are not corpus. A single X post, if later found through one of these locators, may enter a future review packet only after manual human review of author, date, permalink, visible text, attribution, and allowed-use boundaries. No X post can be upgraded directly into a validated method claim.

Public-web locator review note: unauthenticated public web search and public X search access did not return stable specific post URLs during this pass. Candidate leads below therefore remain locator-level leads unless a future human browser review identifies a concrete post permalink.
