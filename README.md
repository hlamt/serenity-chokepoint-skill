# serenity-chokepoint-skill

An Agent Skill for evidence-backed, Serenity-style supply-chain chokepoint investment research, with TOC-informed terminology guardrails.

- [中文说明](README.zh-CN.md)
- [English documentation](README.en.md)

The Skill maps industry value chains, identifies candidate supply-chain chokepoints and companies, and structures evidence, validation paths, risks, tracking indicators, and invalidation conditions.

All outputs are research hypotheses and do not constitute investment advice.
