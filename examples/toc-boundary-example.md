# TOC Boundary Example

## 错误表达

```text
某环节是 AI 产业链的瓶颈。
```

问题在于“瓶颈”没有说明系统边界、目标、有效产出和术语含义，也没有区分供应链卡点、local bottleneck 与 TOC-style constraint。

## 校准表达

```text
更准确地说，该环节可能是当前 AI 产业价值链中的供应链卡点；是否构成 TOC 意义上的瓶颈，需要先定义系统目标、有效产出、系统边界和关键制约资源。
```

## 继续研究

术语校准后仍需完成用户任务：

1. 定义价值链与有效产出。
2. 验证吞吐影响、替代难度、认证与扩产周期。
3. 建立候选公司及其价值捕获证据。
4. 寻找反证并跟踪制约释放和迁移。
5. 写明假设失效条件和免责声明。
