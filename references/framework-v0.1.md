# Framework v0.1

```text
热点 / 产业主题
→ 定义产业价值链
→ 定义有效产出
→ 识别候选供应链卡点
→ 用 TOC-informed CCR 检查
→ 判断价值捕获能力
→ 构建证据链
→ 输出候选公司与投研假设
→ 跟踪制约释放与制约迁移
→ 标记假设失效条件
```

## Stage Gates

1. **产业主题**：写明研究日期、地域、产品边界和时间跨度。
2. **产业价值链**：从投入、关键工艺、交付、应用到最终需求画出节点和依赖。
3. **有效产出**：定义系统真正增加的可交付价值及其可观察指标。
4. **候选卡点**：列出稀缺原因、吞吐影响、替代难度、扩产周期和证据缺口。
5. **CCR 检查**：判断它是系统制约、潜在 CCR、局部问题还是短期短缺。
6. **价值捕获**：检查竞争格局、定价、成本转嫁、客户议价、资本强度与估值预期。
7. **证据链**：按证据等级记录支持、反证、日期、来源与置信度。
8. **候选公司**：解释公司与卡点的暴露路径，不把候选池写成投资结论。
9. **动态跟踪**：监测需求、交期、价格、产能、良率、认证、替代和库存。
10. **失效条件**：给出能明确推翻机制、持续性或价值捕获判断的信号。

## Minimum Decision Rule

A candidate remains a hypothesis until evidence supports system-level throughput impact, persistence, substitution difficulty, credible value capture, and resilience to counterevidence. A strong operational chokepoint can still be a weak investment hypothesis when valuation or bargaining structure captures the benefit elsewhere.
